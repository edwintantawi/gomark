# GoMark
Go Bookmark Rest API

## Objective
Learn to use **GO** by creating a **Bookmark application** Rest API.

This app _aims_ to manage **CRUD** bookmarks

## Tech stack
- [GO language](https://go.dev/)
- [PostgreSQL database](https://www.postgresql.org/)
